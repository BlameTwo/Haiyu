### 功能

##### 数据面板

包含库街区鸣潮数据面板，战双面板正在排队中……

#### 游戏数据

维护鸣潮（国服、国际服、BiliBili）战双（国服、国际服、BiliBili、台服）两个游戏的文件，包含高速下载、快速更新、多线程校验等功能

#### 库街区与游戏

支持库街区登陆（手机号、Token），支持使用账号进行游戏扫码登陆
